# ⚡ AOY TOPUP - ບໍລິການເຕີມເກມອັດຕະໂນມັດ 24 ຊົ່ວໂມງ

ເວັບໄຊບໍລິການເຕີມເກມມືຖື ແລະ ຄອມພິວເຕີ ທີ່ສະດວກ, ວ່ອງໄວ, ທັນໃຈ ແລະ ປອດໄພ 100% ສໍາລັບຜູ້ຫຼິ້ນເກມໃນ ສປປ ລາວ. ລະບົບຖືກອອກແບບດ້ວຍຮູບແບບ Cyberpunk / Dark Mode ທີ່ທັນສະໄໝ ແລະ ພຣີມ່ຽມ.

---

## 🚀 ຄຸນສົມບັດເດັ່ນ (Key Features)

*   **ລະບົບເຕີມເກມ 4 ຂັ້ນຕອນ:** ເລືອກເກມ -> ປ້ອນຂໍ້ມູນບັນຊີ (ID/ເຊີບເວີ) -> ເລືອກແພັກເກດ -> ຊຳລະເງິນ.
*   **ລະບົບອັດຕະໂນມັດ:** ເລືอกຊ່ອງທາງຊຳລະເງິນ BCELOne QR Pay ພ້ອມລະບົບເລືອກຊ່ອງທາງຈ່າຍອັດຕະໂນມັດເມື່ອເລືອກແພັກເກດ.
*   **ປະຫວັດການເຕີມເງິນ:** ບັນທຶກປະຫວັດການເຮັດທຸລະກຳ ແລະ ລະຫັດຄີຕ່າງໆ ລົງໃນ `localStorage` ສາມາດເປີດເບິ່ງໄດ້ຕະຫຼອດເວລາຜ່ານປຸ່ມ "ປະຫວັດການເຕີມ".
*   **ໃບບິນດິຈິຕອນ:** ສະແດງລາຍລະອຽດການຊຳລະເງິນ, QR Code, ບາໂຄດ ແລະ ປຸ່ມກົດຄັດລອກຂໍ້ມູນ (Copy Key) ພ້ອມລະບົບຈຳລອງການຈ່າຍເງິນ.
*   **ອອກແບບຮອງຮັບມືຖື (Mobile Responsive):** ສະແດງຜົນໄດ້ສວຍງາມທັງໃນຄອມພິວເຕີ, ແທັບເລັດ ແລະ ໂທລະສັບມືຖື.

---

## 🛠️ ເຕັກໂນໂລຊີທີ່ໃຊ້ (Tech Stack)

1.  **HTML5:** ໂຄງສ້າງເວັບໄຊ ແລະ Semantic Elements.
2.  **CSS3 (Vanilla CSS):** ການກຳນົດຮູບແບບສີສັນ Cyberpunk, Glassmorphism, Responsive layout ແລະ Animation ຕົວໜັງສືວິ່ງ (Typewriter Effect).
3.  **JavaScript (ES6+):** ການຈັດການຂໍ້ມູນເກມ, ການຄຳນວນລາຄາ, ການກວດສອບຄວາມຖືກຕ້ອງຂອງຟອມ ແລະ ລະບົບຈຳລອງການເຕີມເງິນ.

---

## 📁 ໂຄງສ້າງໂຄງການ (Project Structure)

*   `index.html`: ໜ້າຫຼັກຂອງເວັບໄຊ (Hero, ລາຍຊື່ເກມ, ຈຸດເດັ່ນ, ລີວິວລູກຄ້າ).
*   `topup.html`: ໜ້າຂັ້ນຕອນການເຕີມເກມ ແລະ ຟອມປ້ອນຂໍ້ມູນ.
*   `receipt.html`: ໜ້າໃບບິນແຈ້ງຊຳລະເງິນ ແລະ ປຸ່ມຈຳລອງການໂອນ.
*   `contact.html`: ໜ້າຕິດຕໍ່ສອບຖາມ (WhatsApp, Telegram, Facebook) ພ້ອມຟອມສົ່ງຂໍ້ຄວາມ.
*   `style.css`: ໄຟລ໌ກຳນົດຮູບແບບການອອກແບບ ແລະ ແອນິເມຊັນທັງໝົດ.
*   `script.js`: ຖານຂໍ້ມູນເກມ ແລະ ການເຮັດວຽກຂອງລະບົບ.
*   `images/`: ໂຟລເດີເກັບຮູບພາບກາດເກມ, ໂລໂກ້ ແລະ ໄອຄອນຕ່າງໆ.

## 💻 ວິທີການຕິດຕັ້ງ ແລະ ເປີດໃຊ້ງານ (Installation & Setup)

ໂຄງການນີ້ຖືກພັດທະນາແບບ Static Web ບໍ່ຈຳເປັນຕ້ອງໃຊ້ Server ຫຼັງບ້ານໃນການລັນຕົວຢ່າງ:

1.  ດາວໂຫຼດ ຫຼື Clone ໂຄງການນີ້ລົງເຄື່ອງ:
    ```bash
    git clone https://github.com/itne04/AoyTopup.git
    ```
2.  ເຂົ້າໄປໃນໂຟລເດີໂຄງການ ແລະ ເປີດໄຟລ໌ `index.html` ດ້ວຍບຣາວເຊີ (Chrome, Edge, Safari, Firefox).
3.  ຫຼື ເປີດໃຊ້ງານຜ່ານ Extension **Live Server** ໃນ VS Code ເພື່ອທົດສອບ.
